# 🧠 Emotion Learning Assistant (ELS)

The **Emotion Learning Assistant (ELS)** is an intelligent, AI-powered educational application designed to identify student emotional states during self-directed learning sessions. Using deep learning models, it provides real-time cognitive coping strategies to optimize focus, reduce frustration, and improve overall online learning outcomes.

Developed by **Tithi Goswami**, a third-year BTech student at **Sir CR Reddy College of Engineering**.

## 🛠️ Prerequisites & Installation

Before running the application, ensure you have the following installed on your local system:

*   **Python 3.8+**
*   **Git** (for version control)

### 1. Project Setup
Clone this repository to your local machine and navigate into the source directory:
```bash
git clone [https://github.com/goswamitithi1005-byte/emotion_learning_assistant.git](https://github.com/goswamitithi1005-byte/emotion_learning_assistant.git)
cd emotion_learning_assistant/ELS--Project_CodeFile/src

```

### 2. Set Up Virtual Environment (`venv`)

It is highly recommended to use a virtual environment to manage dependencies:

```bash
# Create the virtual environment (only needed the first time)
python -m venv venv

# Activate the virtual environment
# On Windows:
..\..\venv\Scripts\activate
# On macOS/Linux:
source ../../venv/bin/activate

```

### 3. Install Dependencies

Install all required libraries, including Streamlit, TensorFlow, and other system dependencies:

```bash
pip install -r requirements.txt

```

## 🚀 How to Run the Application

To launch your ELS interface dashboard locally, run the following command sequence in your command prompt:

1. **Navigate to the source directory:**
```cmd
cd C:\ELS\emotion_learning_assistant\ELS--Project_CodeFile\src

```


2. **Activate the environment:**
```cmd
..\..\venv\Scripts\activate

```


3. **Launch Streamlit:**
```cmd
streamlit run app.py

```



Streamlit will automatically host the application and open your default web browser to:
👉 **`http://localhost:8501`**
